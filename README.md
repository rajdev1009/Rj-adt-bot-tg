# Rj-adt-bot-tg
Ek powerful aur customizable Telegram bot jo aapki files ko safe rakhta hai, links generate karta hai, aur unhe shortener ke through monetize karne mein madad karta hai. Isme Admin Dashboard, Premium System, aur Animated UI jaise features inbuilt hain.Force Subscribe: Users ko file lene se pehle aapka channel join karna compulsory hai.
